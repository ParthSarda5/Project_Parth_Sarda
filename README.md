## Project Overview

Binary image classification: does a 64×64 pixel astronomical survey image contain a **strong gravitational lens** or not?

Strong gravitational lensing occurs when a foreground galaxy bends light from a background source, producing arc-like or ring-shaped distortions. This project trains a ResNet-18 CNN to detect these structures automatically.

**Task:** Supervised binary image classification  
**Input:** 64×64 greyscale image of a galaxy field (`.npy` file)  
**Output:** `"Lens"` or `"Non-Lens"` with a probability score (0–1)

## About the Data Format (`.npy`)

The data files use NumPy's `.npy` format — this is the **native raw format** of the dataset because our images are generated by `lenstronomy`, a scientific ray-tracing package that outputs NumPy arrays directly.

**Why `.npy` and not `.jpg`?**
Each image is a float32 array spanning several orders of magnitude (bright galaxy cores vs. faint lensing arcs). Converting to JPEG would clip or quantise this dynamic range and destroy the signal. The submission spec says *"they should be in the form that is found in the raw dataset you have used"* — and the raw form here is `.npy`.

**How to visually inspect the images:**
```python
import numpy as np, matplotlib.pyplot as plt
img = np.load('data/lens_01.npy')          # shape (64, 64), float32
stretched = np.log1p(np.maximum(img, 0))   # log-stretch compresses dynamic range
plt.imshow(stretched, cmap='inferno', origin='lower')
plt.colorbar(); plt.show()
```
A visual preview grid of all 20 example images is saved at **`data/preview.png`** — open this in any image viewer.

## Model

**Backbone:** ResNet-18 (optionally pretrained on ImageNet-1K)  
**Head:** `Linear(512 → 1)` — scalar logit for `BCEWithLogitsLoss`

**Preprocessing (train and inference — identical):**
1. Log-stretch: `log1p(max(x, 0))`
2. Per-image min-max normalisation → [0, 1]
3. Greyscale → 3-channel (repeated)
4. Augmentation (train only): random flips + 180° rotation
5. Normalise: mean=0.5, std=0.5 → [−1, 1]

---

## Data Source

Images are simulated with **lenstronomy** (Birrer & Amara, A&A 2018).

- **Lensed:** SIS lens + Sérsic source + Sérsic lens light + Poisson/Gaussian noise
- **Non-lensed:** Sérsic galaxy, no deflection, same noise model
- **Resolution:** 64×64 px at 0.1 arcsec/pixel

The 20 example files in `data/` are generated by the same `generate_data.py` pipeline as the training data.
